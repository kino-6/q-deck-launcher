## 🎯 標準起動方法

### 開発モード（常にこれを使用）

```powershell
.\launch.ps1 -Force
```

### なぜこれを標準にしたか

1. **ポート競合の自動解決** - 1420-1429の範囲で空きポートを自動検索
2. **プロセスクリーンアップ** - 既存のnode/electronプロセスを自動終了
3. **エラーハンドリング** - 詳細なエラーメッセージと解決策を表示
4. **環境チェック** - Node.jsのバージョン確認
5. **一貫性** - 全ての開発者が同じ方法で起動

### 非推奨のコマンド

以下のコマンドは使用しないでください：

```powershell
# ❌ 使用しない
npm run dev          # Viteのみ、Electron APIなし
npm run electron:dev # ポート管理なし、手動クリーンアップ必要
```
